# Backend

Node.js API gateway scaffold for the chatbot system.

## Layout

- `src/config` environment and runtime configuration
- `src/clients` external service adapters
- `src/controllers` request handlers
- `src/metrics` Prometheus instrumentation
- `src/routes` HTTP route registration
- `src/services` business logic
- `src/state` session storage abstractions
- `src/types` shared request and response contracts
- `src/utils` helper functions
